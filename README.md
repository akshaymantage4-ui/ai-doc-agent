# ai-doc-agent

# 📄 AI Document Assistant

An AI-powered app that allows users to upload documents and ask questions, summarize content, and extract insights.

---

## 🚀 Features

- 📄 Supports multiple file types:
  - PDF
  - DOCX
  - TXT
  - Images (JPG, PNG with OCR)
- 💬 Chat with your documents
- 📊 Extract key insights
- 📝 Summarize documents
- ⚡ Fast processing using vector database

---

## 🛠️ Tech Stack

- Python
- Streamlit
- LangChain
- FAISS
- HuggingFace Embeddings
- Groq LLM

---
